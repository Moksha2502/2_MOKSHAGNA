# Data Directory

Place your DDInter dataset files here.

## Expected Format

The CSV file should have columns for:
- Drug1 / Drug A (first drug name)
- Drug2 / Drug B (second drug name)
- Interaction_Type / Type (e.g., Pharmacokinetic, Pharmacodynamic)
- Severity / Risk (e.g., Major, Moderate, Minor, Contraindicated)
- Mechanism / Mechanism_Description (interaction mechanism)
- Description / Details (detailed description)

If no data file is provided, the application will use built-in sample data.
